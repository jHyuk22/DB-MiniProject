# DB-MiniProject

2023년 1학기 세종대학교 컴퓨터공학과 데이터베이스 수업에서 진행한 미니프로젝트입니다.

## 프로젝트 주제

영화 예매 앱의 데이터베이스 설계

## 프로젝트 설명

사용자들이 영화를 검색하고, 선택한 영화의 티켓을 예매할 수 있는 플랫폼으로, 이를 위해 데이터베이스를 설계하여 영화, 티켓, 회원, 리뷰 등의 정보를 효율적으로 관리함

## 프로젝트 환경

언어: SQL, C++

사용 도구: MSSQL, MFC, Visual Studio
